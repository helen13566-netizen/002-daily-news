## 1. `ripple` 축 — 파급효과

### 적용 프레임워크
- **Meadows Iceberg Model** (4 levels): Events → Patterns → Structures → Mental Models
- **Second/Third-Order Consequences**: "And then what?" 3단계 강제 질문

### Agent 체크리스트
이 축의 text 를 쓸 때 다음을 반드시 포함:
1. **1차 효과**: 이 사건이 **당장** 일으키는 직접 변화 (30초 뒤 일어날 일)
2. **2차 효과**: "그 다음엔?" — 1차 결과가 퍼져 나타날 간접 변화 (며칠~몇 주)
3. **3차 효과 (가능하면)**: 2차 결과가 시스템 균형을 재편할 새 흐름 (수 개월+)
4. **Iceberg 레벨**: 이 사건은 사건(Event) 인가, 패턴(Pattern) 인가, 구조(Structure) 변화인가?

### 예시 (금리 인상)
> 금리가 오르면 대출 이자가 바로 오릅니다(1차). 며칠 지나면 사람들이 소비를 줄이고 기업 매출이 둔화되기 시작해요(2차). 몇 개월 지나 고용이 흔들리면 정부가 금리 되돌림 압력을 받을 수 있습니다(3차). 겉으론 '이자율' 사건처럼 보이지만, 실은 가계·기업·정부 구조(Structure) 차원의 조정이 시작된 신호입니다.

## 2. `history` 축 — 역사·비교

### 적용 프레임워크
- **Reference Class Forecasting** (Kahneman·Tversky): 유사 사례 집합 정의 + 결과 분포
- **Outside View vs Inside View**: 유사 사례 통계 우선

### Agent 체크리스트
1. **Reference class 정의**: 이 사건은 어떤 "사례 집합"에 속하는가? (예: "2020년대 중앙은행 금리 인상 사이클")
2. **그 집합의 실제 결과**: 비슷한 일이 몇 번 있었고, 결과가 어떻게 나왔는가
3. **이번이 어디에 위치**: 평균? 극단? 새로운 국면?
4. **차이점**: 이번과 과거가 다른 변수

### 예시 (호르무즈 긴장)
> 호르무즈 해협 긴장은 지난 20년간 네 번 있었어요(2008 · 2012 · 2019 · 2024). 당시 유가는 평균 한 달 만에 8~12% 올랐고, 두세 달 뒤 원상복귀 했습니다. 이번엔 미·이란 양측 긴장이 과거보다 높아 회복이 좀 더 걸릴 수 있지만, 큰 그림은 비슷한 궤적일 가능성이 높습니다.

## 3. `scenario` 축 — 시나리오

### 적용 프레임워크
- **Tetlock Superforecasting 10 Commandments**: 특히 probabilistic thinking, comparison class, belief updating
- **Probability Grading (5단계)**: 낮음·조금 낮음·반반·조금 높음·높음

### Agent 체크리스트
1. **최선 시나리오** (낮음~중간 확률): 어떻게 일이 좋게 풀리면, 결과는?
2. **최악 시나리오** (낮음~중간 확률): 어떻게 나빠지면, 결과는?
3. **중간 시나리오** (가장 가능성 높음): 가장 있을 법한 전개는?
4. **갈림길**: 다음 며칠~몇 주 안에 **어떤 이벤트가 어느 방향을 결정**하는가
5. **확률 언어**: "~일 수 있다" 가 아니라 "~가능성이 높음/중간/낮음" 으로 구분

### 예시 (기업 M&A 심사)
> 다음 주 공정위 결정에 따라 세 갈래입니다. 무조건 승인(가능성 낮음~중간): 합병 발표 가격대로 가, 단기 주가 +10~15%. 조건부 승인(가장 가능성 높음): 일부 사업 매각 요구, 주가는 보합. 불승인(가능성 낮음): 합병 무산, 주가 -20%+ 충격. 갈림길은 다음 주 금요일 공정위 브리핑입니다.

## 4. `personal` 축 — 개인 조언

### 적용 프레임워크
- **Matt Levine Incentive Analysis**: 누가 이익/손해 → 내 행동
- **Outside View**: 비슷한 상황에서 사람들이 대체로 어떻게 대응했나
- **Action Checklist**: 구체 행동 3개 이하

### Agent 체크리스트
1. **내가 이 뉴스에 노출된 경우**: 어떤 포지션이면 이익/손해? (예: 주담대 차주, 주식 보유자, 자영업자)
2. **구체 행동 2~3개**: 오늘·이번 주·이번 달 단위로 **검토해볼** 만한 행동. (투자 매수·매도 직접 지시는 X, "검토/확인/비교" 수준)
3. **피해야 할 행동 1개**: 반사적으로 하기 쉽지만 이 맥락에선 불리한 행동
4. **정보 팔로업**: 며칠 안에 추가로 볼 지표·발표

### 예시 (금리 동결)
> 주담대 차주시라면 이번 주 안에 현재 금리와 고정금리 갈아탔을 때 월 부담을 비교해보세요. 변동금리를 계속 쓰는 경우 향후 3개월 내 방향성(추가 동결 vs 인하)을 금통위 발표(다음 달 말)까지 관찰해보는 게 좋습니다. 조급하게 전환하거나 추가 대출 늘리는 결정은 피하세요.

## 5. `frame` 축 — 사고 프레임

### 적용 프레임워크
- **Stanford COR 3 Moves**: Who's behind? / What evidence? / What do others say?
- **IMVAIN 5축**: Independent · Multiple · Verifies · Authoritative · Named
- **IFCN 5 Commitments**: Nonpartisanship · Source transparency · Funding transparency · Methodology · Corrections
- **Entman Framing**: 이 기사가 선택한 frame vs 빠진 frame

### Agent 체크리스트
1. **출처 점검 (COR + IMVAIN)**: 이 뉴스 1차 출처는? 단일 출처? 실명? 이해관계?
2. **증거 점검 (COR)**: 기사에 제시된 수치·인용의 실제 근거는? 검증 가능?
3. **교차 확인 (COR)**: 다른 매체는 어떻게 보도? 차이점?
4. **프레임 선택 (Entman)**: 이 기사는 이 사건을 어떻게 정의했나? 빠진 프레임은?
5. **독자 던질 질문 2~3개**: "왜 지금 이 기사가 나왔나?", "반대 진영의 수치는?", "이 주장을 하는 사람의 이해관계는?"

### 예시 (정치 논란 기사)
> 이 기사의 숫자는 한 정부 관계자의 단일 발언이 근거입니다(M·N 약함). 다른 매체 두 곳은 동일 사안을 다르게 숫자로 보도해요 — 교차 확인 필요. '위기' 프레임을 택했는데, 같은 수치를 '회복' 프레임으로 해석할 수 있는 맥락도 있어요. 이 기사 믿기 전 두 질문: 이 관계자의 이해관계는? 정부 공식 통계(KOSIS) 원자료에 가서 확인하면 뭐가 나오나?

## 6. `perspective` 축 — 다관점

### 적용 프레임워크
- **Stakeholder Mapping**: Primary · Secondary · Key 3-tier
- **Entman 4 Functions**: 각 stakeholder 의 Problem/Cause/Moral/Treatment frame
- **Incentive Disclosure**: 각 측의 "이익 vs 손해" 구조

### Agent 체크리스트
1. **Stakeholder 3~5명 식별**: 가장 직접 영향받는 Primary, 간접 Secondary, 결정권 있는 Key
2. **각 stakeholder 의 frame** (한 문장씩): 이 사건을 "어떤 문제"로 보며, 어떤 해결을 원하나
3. **구조적 갈등**: 누가 이익, 누가 손해, 왜 이 지점에서 충돌하나
4. **국제/국내/세대 등 다양한 축**: 최소 2축 이상의 다른 관점

### 예시 (삼성 노조 파업)
> 경영진은 "경쟁력 훼손" 으로 봅니다 — 해결책은 생산 재개. 노조는 "정당한 성과 배분" 으로 봅니다 — 해결책은 성과급 인상. 정부는 "국가 반도체 경쟁력" 으로 봅니다 — 해결책은 조속 합의 + 산업 지원. 소비자·투자자는 "주가·제품 공급" 관점이라 단기 손실이 더 크게 체감됩니다. 세 입장이 부딪히는 지점은 '성과 배분 기준의 투명성' 입니다.

## 축 선택 가이드 (v12 계승 + 프레임워크 반영)

v12 의 카테고리별 축 선택을 유지하되, 각 축의 내용 **품질**이 위 프레임워크를 따르도록 업그레이드.

| 기사 유형 | 기본 2축 | 보너스 축 | 이유 |
|-----------|----------|-----------|------|
| 국제·외교·분쟁 | `ripple` + `history` | `scenario` | 2/3차 연쇄 + reference class + 갈림길 |
| 통화·금리·부동산 | `ripple` + `history` | `personal` | iceberg 구조 + base rate + 행동 체크리스트 |
| 기업 M&A·사업 재편 | `ripple` + `history` | `perspective` + `scenario` | stakeholder 매트릭스 + 확률 |
| AI·기술 신제품 | `ripple` + `history` | `personal` | leverage points + incentive + 내 업무 영향 |
| 재난·안전·의료 | `ripple` + `history` | `personal` | iceberg + 행동 |
| 정치 논란·숫자 모호 | `history` + `frame` | `perspective` | **`ripple` 교체 → `frame`** (COR·IMVAIN·IFCN·Entman) |
| 연예·스포츠·단순 속보 | insights 생략 | — | score ≤ 4 |
